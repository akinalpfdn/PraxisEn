## Tips & Best Practices

### For Effective Learning

#### 1. **Consistency Over Intensity**
- Study 15-20 minutes daily
- Better than 2 hours weekly
- Build a learning habit

#### 2. **Active Recall**
- Try to remember before flipping
- Say the word aloud
- Use in your own sentences

#### 3. **Context Learning**
- Read all example sentences
- Notice word patterns
- Connect to real situations

#### 4. **Review Strategy**
- Don't skip difficult words
- Mark truly known words only
- Review marked as learned weekly

### Study Techniques

#### Before Bed
- Review recently learned words
- Your brain consolidates during sleep
- Improves long-term retention

#### Morning Routine
- Learn new words when fresh
- Better focus and memory
- Sets positive tone for day

#### Using Translation Input
- Test yourself regularly
- Type translations from memory
- Builds active vocabulary

### Common Mistakes to Avoid

#### Don't:
- Rush through cards quickly
- Mark words as known too early
- Skip pronunciation practice
- Ignore example sentences

#### Do:
- Take time with each word
- Review difficult words more
- Practice pronunciation
- Learn from context

### Maximizing Free Version

1. **Use All 20 Daily Words**: Don't waste your limit
2. **Focus on A1-B1 Levels**: Complete free content first
3. **Practice Pronunciation**: No limit on audio
4. **Review Learned Words**: Unlimited access

### When to Upgrade

Upgrade to Premium if you:
- Complete all free levels (A1-B1)
- Want access to B2 vocabulary
- Need unlimited daily practice
- Value advanced learning tools
- Want full offline capability

## Troubleshooting

### Common Issues

#### Audio Not Playing
- Check device volume
- Ensure text-to-speech is enabled
- Restart the app
- Check iOS settings

#### Slow Image Loading
- Check internet connection
- Try again after a moment
- Images cached after first load
- Consider Premium for offline mode

#### Progress Not Saving
- Ensure internet connection for iCloud sync
- Check iOS app permissions
- Try force-quitting and reopening
- Contact support if persists
